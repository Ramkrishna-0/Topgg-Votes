# Topgg-Votes
This repository will help you with tracking votes on top.gg for your Discord Bot.

## Changes
- Open `./index.js`.
- Change the port.
- Change the auth code.
- Enter Token in `.env.example` Rename Your File To .env Then Enter Your token file as :
```
TOKEN="YOUR_TOKEN"
```
